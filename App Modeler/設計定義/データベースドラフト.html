```html テーブル定義

テーブル定義

テーブル名 カラム名 データ型 制約 説明
users user_id INT PRIMARY KEY AUTO_INCREMENT ユーザーの一意識別子
users username VARCHAR(50) UNIQUE NOT NULL ユーザー名
users email VARCHAR(100) UNIQUE NOT NULL メールアドレス
users password_hash VARCHAR(255) NOT NULL ハッシュ化されたパスワード
users created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP アカウント作成日時
products product_id INT PRIMARY KEY AUTO_INCREMENT 製品の一意識別子
products name VARCHAR(100) NOT NULL 製品名
products description TEXT 製品説明
products price DECIMAL(10,2) NOT NULL 製品価格
products stock_quantity INT NOT NULL 在庫数量
products created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP 製品登録日時
categories category_id INT PRIMARY KEY AUTO_INCREMENT カテゴリの一意識別子
categories name VARCHAR(50) UNIQUE NOT NULL カテゴリ名
categories description TEXT カテゴリ説明
product_categories product_id INT FOREIGN KEY REFERENCES products(product_id) 製品の一意識別子
product_categories category_id INT FOREIGN KEY REFERENCES categories(category_id) カテゴリの一意識別子
orders order_id INT PRIMARY KEY AUTO_INCREMENT 注文の一意識別子
orders user_id INT FOREIGN KEY REFERENCES users(user_id) ユーザーの一意識別子
orders total_amount DECIMAL(10,2) NOT NULL 注文の合計金額
orders status VARCHAR(50) NOT NULL 注文のステータス
orders created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP 注文日時
order_items order_item_id INT PRIMARY KEY AUTO_INCREMENT 注文アイテムの一意識別子
order_items order_id INT FOREIGN KEY REFERENCES orders(order_id) 注文の一意識別子
order_items product_id INT FOREIGN KEY REFERENCES products(product_id) 製品の一意識別子
order_items quantity INT NOT NULL 注文数量
order_items price DECIMAL(10,2) NOT NULL 注文時の製品価格
cart_items cart_item_id INT PRIMARY KEY AUTO_INCREMENT カートアイテムの一意識別子
cart_items user_id INT FOREIGN KEY REFERENCES users(user_id) ユーザーの一意識別子
cart_items product_id INT FOREIGN KEY REFERENCES products(product_id) 製品の一意識別子
cart_items quantity INT NOT NULL カート内の製品数量
cart_items added_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP カートに追加された日時
reviews review_id INT PRIMARY KEY AUTO_INCREMENT レビューの一意識別子
reviews product_id INT FOREIGN KEY REFERENCES products(product_id) 製品の一意識別子
reviews user_id INT FOREIGN KEY REFERENCES users(user_id) ユーザーの一意識別子
reviews rating INT NOT NULL 製品の評価（1-5）
reviews comment TEXT レビューコメント
reviews created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP レビュー作成日時
```
